# 🧠 Fashion-MNIST Image Classification using Artificial Neural Networks

This project marks my first step into Deep Learning. Using TensorFlow and Keras, I built an Artificial Neural Network (ANN) capable of classifying Fashion-MNIST images into 10 clothing categories.

## 🚀 Features
- Data Preprocessing & Normalization
- Fashion-MNIST Dataset Analysis
- ANN Architecture Design
- Model Training & Evaluation
- Accuracy & Loss Visualization
- Fashion Item Prediction

## 🛠️ Tech Stack
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Jupyter Notebook

## 📊 Dataset
Fashion-MNIST consists of 70,000 grayscale images (28×28 pixels) across 10 fashion categories including shirts, trousers, sneakers, bags, and more.
